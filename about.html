About WeatherWise

WeatherWise is a weather app designed to help users quickly check weather information and get useful recommendations for planning their day.

The idea behind the project

The app retrieves weather data for the city and, based on that data, provides users with recommendations regarding, for example, what to wear, outdoor activities and preparedness. The goal is to create a clear and user-friendly web app that works on both computers and mobile devices.

Key Features

  • Checking the city's weather
  • Displaying current weather information
  • Reviewing the weather forecast
  • Weather-based recommendations for users
  • Saving favorite cities to localStorage
  • Responsive user interface

Technologies used

  • HTML
  • CSS
  • JavaScript
  • Fetch API
  • LocalStorage

Data source

The app uses open weather data from an external API. Weather data is retrieved using JavaScript and displayed to the user in a clear format.

The goal of the exercise

The goal of this project is to develop a multi-page, responsive and functional web application that utilizes JavaScript, local storage and external data.

Developer

This project was carried out as part of a web programming assignment.
